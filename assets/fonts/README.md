# Fonts Directory

This directory is for custom font files (optional).

## Current Status

The app is currently configured to use **Lexend Deca** font family, but font files are **optional**. If font files are not present, the app will automatically use system fonts, which works perfectly fine.

## To Add Custom Fonts

1. Download Lexend Deca font from: https://fonts.google.com/specimen/Lexend+Deca
2. Extract the font files and place the following `.ttf` files in this directory:
   - `LexendDeca-Regular.ttf`
   - `LexendDeca-SemiBold.ttf`
   - `LexendDeca-Bold.ttf`

3. Restart your development server after adding fonts

## Note

The app will work perfectly fine without these font files - it will use the system default fonts instead.
